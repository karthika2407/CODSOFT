# CODSOFT - Java Development Internship

## Task 1 - Number Guessing Game

A simple Java game where the user has to guess a randomly generated number. The program provides hints such as "Too High" or "Too Low" until the correct number is guessed.

## Task 2 - Student Grade Calculator

A Java program that takes marks as input, calculates the total marks and average percentage, and displays the grade.

## Task 3 - ATM Interface

A basic ATM simulation developed in Java that allows users to check balance, deposit money, and withdraw money through a menu-driven interface.

* Java
